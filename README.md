Global Superstore Sales & Profit Analysis
Project Overview

This project analyzes the Global Superstore dataset to understand sales performance, profit trends, customer behavior, discount impact, and regional contribution.

The objective was to uncover actionable business insights that help improve profitability, optimize discount strategies, and identify high-performing customers and regions.

The complete project was executed using:

Python for data cleaning and preprocessing

SQL for business analysis and querying

Power BI for interactive dashboard visualization

This project follows a complete end-to-end data analytics workflow.

Key Insights

Certain sub-categories consistently generate negative profits despite strong sales.

Higher discounts generally correlate with reduced profitability.

A small group of customers contribute disproportionately to total profit.

Some regions outperform others significantly in profit margin.

Delivery performance fluctuates seasonally.

Not all high-sales products generate high profit.
